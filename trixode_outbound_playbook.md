This playbook is built for a founder-led outbound motion (Hussien as the only seller), no paid ads, and a local-service-business-first strategy.

## 3) Outbound Script System - Hook, Meat, CTA

Use this structure in every cold message:

1) Hook (first 2 lines) = PAS Problem + Agitation
2) Meat (middle 2-3 lines) = Solve + proof + relevance
3) CTA (last 1-2 lines) = low-friction next step

## 3.1 Hook templates (first 2 lines)

| Hook type | PAS angle | Fill-in template |
|---|---|---|
| Website audit | Problem + Agitate | "[FIRST_NAME], noticed [COMPANY_SITE] loads in [LOAD_TIME]s on mobile and the main [CTA_ELEMENT] is buried. That usually costs [VERTICAL] firms in [CITY] a chunk of high-intent leads each week." |
| Competitor gap | Problem + Agitate | "[FIRST_NAME], saw [COMPETITOR_1] and [COMPETITOR_2] now rank above [COMPANY] for [KEYWORD]. If that trend holds, [IMPACT_METRIC] gets harder and more expensive every month." |
| Trigger event | Problem + Agitate | "Congrats on [TRIGGER_EVENT], [FIRST_NAME]. Most teams hit a lead-handling bottleneck right after that move, especially when [CURRENT_SYSTEM] is still manual." |
| Missing feature | Problem + Agitate | "Quick one, [FIRST_NAME] - I could not find [MISSING_FEATURE] on [COMPANY_SITE]. For [VERTICAL] buyers, missing that step often means they bounce to [COMPETITOR_OPTION]." |
| Review reference | Problem + Agitate | "Read a few recent reviews for [COMPANY], and a pattern showed up around [REVIEW_THEME]. If prospects keep seeing that before booking, conversion usually dips even with strong traffic." |
| Local market shift | Problem + Agitate | "In [CITY], search demand for [SERVICE] jumped this [TIME_PERIOD], but [COMPANY] is still positioned for [OLD_POSITIONING]. That mismatch usually leaves easy wins on the table." |

## 3.2 Meat templates (middle 2-3 lines)

| Meat type | Fill-in template |
|---|---|
| Result-led | "We help [VERTICAL] businesses turn traffic into booked [APPOINTMENTS_OR_JOBS] by fixing [FIX_1], [FIX_2], and [FIX_3]. For [SIMILAR_COMPANY], this translated into [RESULT_METRIC] in [TIMEFRAME]." |
| ROI-led | "If [COMPANY] improves conversion from [CURRENT_CONV]% to [TARGET_CONV]%, at [MONTHLY_TRAFFIC] visits that is about [EXTRA_LEADS] extra leads/month. At [CLOSE_RATE]% close rate and [AVG_JOB_VALUE] CAD average value, that is roughly [REVENUE_LIFT] CAD/month." |
| Competitor-led | "Right now [COMPETITOR] is winning on [COMPETITOR_EDGE]. We usually close that gap first with [TACTIC_1], then add [TACTIC_2] so your site captures demand instead of leaking it." |
| Specificity-led | "I recorded [NUMBER] specific fixes for [COMPANY]: [FIX_1], [FIX_2], [FIX_3]. None require a full rebuild to start improving lead flow." |
| AI teaser | "We also deploy a lightweight [AI_COMPONENT] that handles [FOLLOW_UP_TASK] so hot leads do not go cold after hours. It saves [TEAM_ROLE] from manual chasing and lifts response speed." |
| Risk-reversal-led | "We run implementation in milestones with [MILESTONE_NAME_1] then [MILESTONE_NAME_2], so you see progress before full rollout. If [GUARANTEE_CONDITION], we keep working at no extra fee until it is fixed." |

## 7) Discovery Call Script (15 minutes)

## 7.1 15-minute flow

| Segment | Time | Objective |
|---|---:|---|
| Opening | 2 min | Set agenda + permission |
| Diagnosis | 5 min | Understand pain, economics, urgency |
| Pivot | 3 min | Match offer to problem with ROI frame |
| Close | 3 min | Investment, process, next step |
| Buffer | 2 min | Questions and scheduling |

## 7.2 Opening script (2 minutes)

"[FirstName], thanks for making time. I suggest we do this in three parts: first, I will ask a few questions to understand your current lead flow and bottlenecks. Second, if there is a fit, I will outline the fastest path we would recommend. Third, we will decide if it makes sense to move forward now, later, or not at all. Sound good?"

## 7.3 Diagnosis questions (5 minutes)

| Question template | Why this question matters |
|---|---|
| "How are new leads currently coming in today - [CHANNEL_1], [CHANNEL_2], or mostly referrals?" | Maps channel dependency and risk |
| "Roughly how many leads do you get monthly, and how many become paying clients?" | Establishes conversion baseline |
| "What is your average customer value in CAD for [SERVICE]?" | Enables ROI math |
| "Where do you think leads are leaking right now - traffic, conversion, or follow-up?" | Identifies bottleneck to solve first |
| "What have you already tried in the last [TIMEFRAME], and what happened?" | Avoids repeating failed approaches |
| "If we fix one thing in the next [30-60] days, what outcome matters most?" | Defines dream outcome + urgency |
| "Who besides you needs to approve this decision?" | Clarifies authority and next steps |

## 7.4 The pivot (3 minutes): offer + ROI framing

### Pivot script template

"Based on what you shared - [PAIN_SUMMARY] and a goal of [OUTCOME_TARGET] - the best fit is [OFFER_NAME]. We would focus first on [LEVER_1], [LEVER_2], and [LEVER_3], then layer [SEO_OR_AI_COMPONENT] so gains stick after launch."

### ROI formula to use live

| Metric | Formula | Placeholder example |
|---|---|---|
| Extra leads/month | `[MONTHLY_TRAFFIC] x ([TARGET_CONV] - [CURRENT_CONV])` | `[EXTRA_LEADS]` |
| Extra customers/month | `[EXTRA_LEADS] x [CLOSE_RATE]` | `[EXTRA_CUSTOMERS]` |
| Revenue lift/month | `[EXTRA_CUSTOMERS] x [AVG_JOB_VALUE_CAD]` | `[MONTHLY_LIFT_CAD]` |
| Payback period | `[INVESTMENT_CAD] / [MONTHLY_LIFT_CAD]` | `[PAYBACK_MONTHS]` |

## 7.5 The close (3 minutes)

"If we move forward, investment for [OFFER_NAME] is [PRICE_CAD]. Process is simple: [STEP_1], [STEP_2], [STEP_3]. Timeline is [TIMELINE]. If this feels aligned, I can send the proposal today and we can start [START_DATE]."

## 7.6 "Need to think about it" response

"Totally fair. Usually that means one of three things: timing, budget, or confidence in the result. Which one is the real blocker for [Company] right now? If helpful, I can send a one-page decision brief with ROI, scope, and timeline so you can review it with [PARTNER_OR_TEAM]."

## 8) Objection Handling Scripts (Acknowledge -> Clarify -> Reframe -> Proof -> Close)

## 8.1 "It is too expensive" (Price)

| Step | Script template |
|---|---|
| Acknowledge | "Makes sense, [FirstName] - you should protect budget and be sure this pays back." |
| Clarify | "When you say expensive, are we comparing to your current option, a different agency, or cash-flow timing this month?" |
| Reframe | "The real comparison is not [PRICE_CAD] vs zero. It is [PRICE_CAD] vs the cost of missed leads over [TIMEFRAME]." |
| Proof | "For [SIMILAR_COMPANY], improving conversion by [DELTA]% added about [MONTHLY_LIFT_CAD] monthly pipeline, which covered the project in [PAYBACK_MONTHS] months." |
| Close | "If we structure this as [PAYMENT_OPTION], does that make the decision workable?" |

## 8.2 "Not right now" (Timing)

| Step | Script template |
|---|---|
| Acknowledge | "I hear you - timing can kill good projects if it is wrong." |
| Clarify | "Is the constraint bandwidth, budget timing, or priority versus other initiatives?" |
